# uniapp-vuecli3

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
(需完善mock.config.js dev.config.js 请求前缀才能正常启动)
npm run mock:mp-weixin 启动mock环境
npm run dev:mp-weixin 启动dev环境

```

### Compiles and minifies for production
```
npm run build
```
